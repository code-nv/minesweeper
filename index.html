    1. The grid contains a random number of mines.
    2. Clicking a tile will reveal what's hiding underneath.
    3. If a tile is adjacent to a mine, it will display how many mines it is adjacent to.
    4. If the tile is not adjacent to any mines, it will reveal all of its adjacent tiles until the revealed tiles are adjacent to mines.
    5. You can mark tiles believed to be mines by toggling a flag. Flagged tiles will not reveal themselves and will be protected from accidental clicks.
    6. The game ends when all safe tiles have been revealed.
